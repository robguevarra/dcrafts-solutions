# Dcrafts Operations Platform

> Centralized order management, real-time designer KDS, and AI-powered customer service for Dcrafts — a custom paper-cut letter business selling on TikTok Shop and Shopee.

---

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:3000/admin/orders
```

Requires `.env.local` with credentials. See [`docs/environment.md`](./docs/environment.md).
